# Sabina
Автор программы: Саврикова Сабина
# Программа представляет собой десктоп‑приложение на Python с графическим интерфейсом (GUI), созданное с использованием библиотеки tkinter. Её основная функция — генерация случайных задач из предопределённого списка и ведение истории выполненных заданий с возможностью фильтрации.

Основные функции
Генерация случайных задач

Выбирает случайную задачу из списка PREDEFINED_TASKS.

Отображает выбранную задачу в интерфейсе.

Добавляет задачу в историю и сохраняет её в файл tasks.json.

Ведение истории задач

Сохраняет все сгенерированные и добавленные задачи в файл tasks.json в формате JSON.

Загружает историю при запуске программы.

Отображает историю в виде списка с указанием названия и категории задачи.

Фильтрация истории

Позволяет фильтровать отображаемую историю по категориям: «учёба», «спорт», «работа», «другое».

Есть опция «все» для отображения полной истории.

Добавление новых задач

Пользователь может добавить собственную задачу с указанием названия и категории.

Новая задача сразу попадает в список для генерации и в историю.

Сохранение данных

История задач сохраняется между запусками программы.

Используется JSON‑файл для структурированного хранения данных.

Обработка ошибок

Проверка на пустой список задач перед генерацией.

Валидация ввода при добавлении новой задачи (непустое название).

Обработка ошибок при чтении/записи файла истории.

Интерфейс пользователя
Заголовок «Ваша задача:» — текстовая метка.

Отображение текущей задачи — крупная надпись с названием задачи.

Кнопка «Сгенерировать задачу» — запускает выбор случайной задачи.

Фильтр по типу — выпадающий список для выбора категории фильтрации истории.

История — прокручиваемый список задач с отображением названия и категории в формате «Название (категория)».

Поле ввода новой задачи — текстовое поле для ввода названия задачи.

Выпадающий список категорий для новой задачи: «другое», «учёба», «спорт», «работа».

Кнопка «Добавить задачу» — добавляет новую задачу в систему.

Технические детали
Язык программирования: Python 3.x.

Библиотека GUI: tkinter (встроенная в Python).

Хранение данных: файл tasks.json (формат JSON).

Кодировка: UTF‑8 (поддержка кириллицы).

Размер окна: 450×500 px, фиксированный.

Примеры использования
Пример 1. Генерация случайной задачи

Запустите программу.

Нажмите кнопку «Сгенерировать задачу».

В поле отображения появится одна из предопределённых задач, например: «Прыжки на скакалке».

Эта задача добавится в историю с категорией «спорт».

Повторите несколько раз — в истории появятся разные задачи из списка.

Пример 2. Фильтрация истории по категории

Сгенерируйте несколько задач разных категорий.

В выпадающем списке «Фильтр по типу» выберите «спорт».

В списке истории останутся только задачи с категорией «спорт», например:

«Сдать норматив (спорт)»;

«Прыжки на скакалке (спорт)».

Выберите «все», чтобы увидеть полную историю.

Пример 3. Добавление собственной задачи

В поле «Добавить задачу» введите название, например: «Прочитать главу книги».

Выберите категорию из выпадающего списка, например: «учёба».

Нажмите кнопку «Добавить задачу».

Новая задача появится в истории и будет доступна для генерации.

Сгенерируйте задачу — есть шанс, что выпадет ваша новая задача.

Пример 4. Работа с сохранённой историей

Добавьте несколько задач и сгенерируйте их.

Закройте программу.

Перезапустите программу.

Обратите внимание, что история задач сохранилась — все ранее сгенерированные и добавленные задачи отображаются в списке.

Попробуйте отфильтровать историю — фильтр работает так же, как и до закрытия программы.
